# 浙江大学新生踩坑指北
## 前言
Hi，你好，浙大人

这里是你们的学长组，欢迎来到“浙里”

又是一年迎新季，稚气未脱的你们，带着对未来的无限憧憬，在美丽的求是园相遇

或许，这是你第一次离开父母；或许，这一次不再有熟悉的朋友陪伴在身旁

但是，不要担心，这一路上，会有那么一群人

伴你同行，陪你成长

让你在求是路上不惧孤独

在创新途中坚定前行
